# Vika Love — одностраничное признание

Минималистичный одностраничный сайт‑признание в любви (Вика + Даня) на Next.js + Tailwind CSS + Framer Motion.

## Запуск

1) Установи Node.js (рекомендуется LTS 20+).
2) В корне проекта:

```bash
npm install
npm run dev
```

Открой `http://localhost:3000`.

## Фото Вики

Положи 4 фото в `public/vika/` и назови:

- `vika-1.jpg`
- `vika-2.jpg`
- `vika-3.jpg`
- `vika-4.jpg`

Если имена/расширения другие — обнови массив `photos` в `src/app/page.tsx`.
